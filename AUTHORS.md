## Acknowledgments

This package integrates the Oat UI library created by Kailash Nadh into the R Shiny ecosystem. The shinyoats package provides R wrappers, Shiny bindings, and integration layer, while all Oat UI components, styling, and JavaScript behaviors are the work of Kailash Nadh.
